# Contributing

Bug reports, issues, and pull requests are welcome.

For general contribution guidelines, see
[CONTRIBUTING.md](https://github.com/cmakefmt/cmakefmt/blob/main/CONTRIBUTING.md)
in the main `cmakefmt` repository.

## This Repo

This repository contains the official Homebrew tap for `cmakefmt`. The formula
is automatically rendered and pushed here by the main release workflow on each
tagged release — manual edits to `Formula/cmakefmt.rb` will be overwritten on
the next release.

If you spot a bug in the formula template, open an issue or PR against the
main `cmakefmt` repository (the template lives in `scripts/render-homebrew-formula.sh`).
